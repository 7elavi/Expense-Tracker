# Expense Tracker

**Автор:** [Максимов Илья]

## Описание программы
Приложение для отслеживания личных расходов. Позволяет добавлять записи с указанием суммы, категории и даты, сохранять их в JSON-файл и фильтровать по параметрам.

## Функционал
- **Добавление расходов:** валидация суммы (положительное число) и даты (формат YYYY-MM-DD).
- **Хранение данных:** автоматическое сохранение и загрузка из `expenses.json`.
- **Фильтрация:** по категориям и временным периодам.
- **Статистика:** автоматический подсчет суммы за выбранный период.

## Требования
Для работы приложения необходим Python версии 3.x. Внешние зависимости отсутствуют (используются стандартные модули `tkinter`, `json`, `datetime`).

## Инструкция по запуску
1. Склонируйте репозиторий.
2. Запустите файл командой: `python main.py`.

## Как использовать
1. Запустите `main.py`.
2. Введите сумму (число), выберите категорию и укажите дату (ГГГГ-ММ-ДД).
3. Нажмите "Добавить". Данные сохранятся в файл `expenses.json`.
4. Для поиска используйте блок "Фильтр", указывая категорию или диапазон дат.

## Примеры использования (Тесты)
* **Тест 1 (Позитивный):** Ввод суммы "500", выбор "Еда", дата "2026-04-27". Результат: запись добавлена, итог увеличился на 500.
* **Тест 2 (Валидация):** Ввод суммы "-100" или "текст". Результат: программа покажет ошибку "Сумма должна быть положительным числом".
* **Тест 3 (Дата):** Ввод даты "27.04.2026". Результат: ошибка "Формат даты: ГГГГ-ММ-ДД".
* **Тест 4 (Фильтр):** Выбор категории "Транспорт". Результат: в таблице отображаются только расходы на транспорт, "Итого" пересчитывается только для них.
